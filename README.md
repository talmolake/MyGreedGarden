# My Green Garden

## Overview

Plant Care is a mobile application designed to help gardening enthusiasts manage and access plant care information.

The application provides users with plant guides containing important details such as sunlight requirements, watering frequency, soil type, and common pests. Users can personalise their experience by saving favourites and creating their own plant entries.

## Features

- Browse plant care guides
- Search, sort, and filter plant information
- View detailed plant profiles
- Add plants to favourites
- Create custom plant guides
- Remove saved favourites
- Watering conversion calculator
- Customisable preferences:
  - Dark mode
  - Font size
  - Typeface settings

## Tech Stack

- JavaScript
- Firebase
- Firebase Firestore
- Node.js
- React Native
